# CSS属性

## 字体、文本

1. font-size：字体大小
2. color：文本颜色
3. text-align：对齐方式
4. line-height：行高

## 背景

1. background：背景颜色

## 边框

1. border：设置边框，符合属性

## 尺寸

1. width：宽度
2. height：高度

## 盒子模型

1. margin：外边距
2. padding：内边距
   1. 默认情况下内边距会影响整个盒子的大小
   2. box-sizing，border-box：设置盒子的属性，让width和height就是最终盒子的大小
3. float：浮动
   1. left
   2. right
