# 系统分区方案

## 通用分区方案
/boot分区：引导分区 200M 

swap分区：交换分区 当内存不够用的时候，将硬盘中的空间充当内存使用

    内存<8G   建议交换分区=内存大小的1.5倍
    
    内存>=8G   建议内存大小为8G
    
/ ：根分区，剩余的全部空间

## 数据信息比较重要的分区方案

/boot 200M

swap 

    <8G 1.5倍内存大小 
    
    >=8G  8G
 
/ 20G~200G

/date 剩余全部空间
